# 실행과 설정

## 실행

macOS에서는 프로젝트 루트의 `Runasis.command`를 더블클릭해 실행합니다. 이 스크립트는 Runasis 서버를 시작하고 브라우저를 여는 사용자용 진입점입니다.

터미널에서는 다음 명령을 사용합니다.

```bash
npm start
```

기본 URL은 보통 `http://localhost:3000`입니다. `3000` 포트가 이미 사용 중이면 다음 사용 가능한 포트를 시도합니다.

## Strava 연결

1. Strava의 `Settings > My API Application`으로 이동합니다.
2. API application을 만들거나 기존 application을 엽니다.
3. `Authorization Callback Domain`을 `localhost`로 설정합니다.
4. Runasis를 실행합니다.
5. 앱에서 Client ID와 Client Secret을 저장합니다.
6. `Connect Strava`를 눌러 Strava 승인 화면을 통과합니다.
7. `Sync`를 눌러 활동 목록과 best effort 세부 정보를 가져옵니다.

큰 활동 이력은 동기화가 오래 걸릴 수 있습니다. 세부 정보는 batch 단위로 가져오므로, 앱이 남은 record를 표시하면 `Sync`를 다시 실행합니다.

## macOS app wrapper

브라우저 대신 앱 창으로 사용하고 싶을 때만 wrapper를 빌드합니다.

```bash
scripts/build-macos-app.sh
```

빌드 후 프로젝트 폴더의 `Runasis.app`을 Finder에서 엽니다. 앱 창을 닫으면 로컬 서버도 중지됩니다. 재빌드에는 Xcode Command Line Tools가 필요합니다.

## 로컬 설정과 민감 경로

- `.env`: Strava API 설정을 저장하는 로컬 파일입니다. 내용은 vault에 기록하지 않습니다.
- `data/strava/activities/details/`: UI가 쓰는 축약된 상세 record 경로입니다.
- `data/strava/activities/raw-details/`: Strava 상세 activity 원본 payload 저장 경로입니다.
- `data/strava/activities/raw-streams/`: activity stream 원본 저장 경로입니다.

위 경로는 구조 이해용으로만 문서화합니다. 실제 파일 내용은 읽거나 노트에 복사하지 않습니다.
